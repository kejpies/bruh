# bruh

Bruh sound, but as a program.

## Installation

```bash
git clone https://github.com/kejpies/bruh.git
cd bruh
make
sudo make install
```
### AUR
If you use Arch Linux you can build the package using [AUR](https://aur.archlinux.org/packages/bruh/).

## Uninstalling

```bash
sudo make uninstall
```

## Usage

```bash
bruh
```
